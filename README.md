## Game Controls
	Oculus Rift
		Teleport -------------- Joystick
		Shoot ----------------- Trigger
		Interact with UI ------ Trigger
		Pick up and grop guns - Grip
	HTC Vive
		Teleport -------------- Trackpad
		Shoot ----------------- Trigger
		Interact with UI ------ Trigger
		Pick up and drop guns - Grip
